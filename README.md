## Instalación y setup local

```bash
# Clona el repositorio
git clone https://github.com/Pikaslel/HAD-Dev-Store.git

# Entra a la carpeta del repositorio clonado
cd tu-repositorio
```

> Para previsualizar el tema en local se requiere acceso a una tienda de Shopify
> y tener [Shopify CLI](https://shopify.dev/docs/themes/tools/cli) v3 o superior
> y [Node.js](https://nodejs.org) v20.10 o superior instalados.
> La navegación y revisión del código puede hacerse directamente desde este repositorio.

## ¿Cómo navegar este repositorio?

- `/docs` → Documentación del proyecto
- `/sections` → Secciones personalizadas del tema
- `/snippets` → Snippets reutilizables
- `/assets` → Archivos JS y CSS
